In today's session I gained first insights into the world of programming. In the following paragraphs the main points of the lecture are briefly summarized.

The Web
The Web is a collection of HTML (HyperText Markup Language) documents. It was developed in the 1990s and currently comprises about 30bn pages.
Through web browsers like (Firefox or Chrome) personal computers operated by end users can send specific requests to servers which then return HTML documents. The global network consisting of (among others) computers and servers is called the Internet.

Features of HTML
The syntax of HTML comprises three key components.

1. Tags specify the content.
2. Elements mostly consist of an opening and closing tag (exception: "Void tags") and content inbetween.
3. Attributes may prompt additional operations.

Additional information
Computers incapacity to cope with even the slightes deviations from formal code requires programmers to be highly accurate in their commands. In doing so they can however rely on a vast amount of online sources.
